Use-Case and Interaction-Flow

Use Case

Bob is looking for a job and wants to join Amarillo TechHub

Precondition: Bob is on the Tech Hub home page

Post-condition: Bob has created his profile

  • Bob clicks on the create profile button
  • A modal appears with form containers
  • He fills them out and clicks submit
  • The site emails him with a verification code
  • Bob checks his email and copies the code
  • Bob enters the code and clicks submit
  • The window closes and Bob is now logged in to his account

Bob has now created his profile
